paramiko3k by varialus

py3k branch of native python SSHv2 protocol library

Goals

  1. Support the latest stable Python 3.x.
  2. Upload package to PyPI.
  3. Support Python 3.0+
  4. Support Python 2.2+.
  5. Merge upstream.

Dependencies

Latest stable pycrypto.

Installation

Debian Wheezy

  1. The program sudo runs commands with root privileges. If you don't have sudo set up, you should be able to use the program su in a separate command to switch to the root user account.

    su -

  2. Install python3-dev from the main Debian repository.

    sudo apt-get install python3-dev

  3. I'm not sure whether build-essential from the main Debian repository is required to compile pycrypto, but I had it installed.

    sudo apt-get install build-essential.

  4. Using Python 3.x run the PIP install script from the web.

    wget https://raw.github.com/pypa/pip/master/contrib/get-pip.py
    sudo python3.2 ./get-pip.py

  5. Use pip-3.2 to install pycrypto from PyPI. Once paramiko3k has been uploaded to PyPI or merged with paramiko, pycrypto will be installed as a dependency during the next step.

    sudo pip-3.2 install pycrypto

  6. Until paramiko3k has been uploaded to PyPI or merged with paramiko, download, unzip and import paramiko3k from github.

    wget https://github.com/varialus/paramiko3k/zipball/py3kb
    unzip paramiko3k
    ls
    cd varialus-paramiko3k-...
    python3.2
    import paramiko3k

  7. After paramiko3k has been uploaded to PyPI or merged with paramiko, which hasn't occurred as of March 2012, use pip to install paramiko3k or paramiko.

    sudo pip-3.2 install paramiko3k
    sudo pip-3.2 install paramiko
    python3.2
    import paramiko3k

Windows

  1. On Windows XP or Windows 7 32-bit, install the 32-bit version of Python 3.2. On Windows 7 64-bit, I recommend installing the 64-bit version of Python 3.2, but the 32-bit version should also work.
  2. Add C:\Python32 and C:\Python32\Scripts to your path.

    Start > Contextual/Right Click Computer > Properties > Advanced > Environment Variables > Path

  3. Install the distribute version of Easy Install.

    http://python-distribute.org/distribute_setup.py python3.2 distribute_setup.py

  4. Install pip.

    easy_install pip

  5. Install and configure Microsoft Visual Studio 2008 Express.

    http://bit.ly/H7o2gp

  6. Use pip-3.2 to install pycrypto from PyPI. Once paramiko3k has been uploaded to PyPI or merged with paramiko, pycrypto will be installed as a dependency during the next step.

    cd C:\Python32\Scripts pip-3.2.exe install pycrypto

  7. Until paramiko3k has been uploaded to PyPI or merged with paramiko, download, unzip and import paramiko3k from github.

    https://github.com/varialus/paramiko3k/zipball/py3kb
    Unzip paramiko3k
    Open command prompt.
    cd C:\path\to\varialus-paramiko3k-...
    python3.2
    import paramiko3k

  8. After paramiko3k has been uploaded to PyPI or merged with paramiko, which hasn't occurred as of March 2012, use pip to install paramiko3k or paramiko.

    cd C:\Python32\Scripts
    pip-3.2.exe install paramiko3k
    pip-3.2 install paramiko3k
    python3.2
    import paramiko3k
